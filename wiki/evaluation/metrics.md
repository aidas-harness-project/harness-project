[Evaluation Harness](../agents/evaluation-harness.md)가 측정한다. 정답지는
모델에 숨겨둔 실제 최종 손사서와 지급/감액 회복 결과다.

# 정량 기준

| 항목 | 3주 PoC 목표 | 평가 방식 |
| --- | --- | --- |
| 핵심항목 추출 정확도 | 80% 이상 | 필드별 정답 일치율 |
| 청구담보 추출 정확도 | 75% 이상 | 정답 담보 포함 여부 |
| 감액사유 Top-3 일치율 | 75% 이상 | Top-1 / Top-3 일치율 |
| 사건 유형 분류 정확도 | 80% 이상 | 분류 정확도 |
| 약관 조항 Top-3 포함률 | 60~70% 이상 | 관련 조항 Top-3 포함 여부 |
| 손사서 초안 생성 성공률 | 70% 이상 | 실제 수정 가능한 수준인지 |
| 케이스당 처리 시간 | 10분 이내 | — |
| 문서 간 불일치 | — | 전문가가 중요하다고 본 불일치 탐지 여부 |
| 반박 포인트 | — | 손사 평가 1~5점 |
| 의료 포인트 | — | 의사 평가 1~5점 |
| 시간 절감 | — | 기존 작성 시간 대비 예상 절감률 |

# 정성 기준

| 평가자 | 기준 |
| --- | --- |
| 손해사정사 | 쟁점이 실무적으로 맞는가 |
| 손해사정사 | 약관 매핑이 검토 출발점으로 쓸 만한가 |
| 손해사정사 | 반박 논리가 실제 손사서에 반영 가능한가 |
| 의사(교수) | 의료적 표현이 과장되지 않았는가 |
| 의사(교수) | 의무기록 기반 검수 포인트가 잘 잡혔는가 |

판정으로 이어지는 조건은 [Go/No-Go 기준](go-no-go.md) 참고.
