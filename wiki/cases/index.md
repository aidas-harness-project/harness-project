# 케이스

가명처리된 독립손해사정사 종결 케이스. PoC의 모델 입력이며, 각
케이스에 포함된 최종 손해사정서는 정답지로서 모델에는 숨긴다.
(후유장해 폴더는 2026-07-07 intake 검증에서 별개 사고 2건으로 분리)

* [골다공증 기여도 감액 케이스](preexisting-condition.md) - 기왕증·퇴행성 기여도로 장해지급률이 감액된 사례 (R01).
* [뇌혈관질환진단비 분쟁 케이스](coverage-dispute.md) - 약관상 지급범위를 두고 4개 보험사와 다툰 사례 (R04/R05).
* [상완골 골절 후유장해 케이스 (CASE_003)](permanent-disability.md) - 보도블럭 전도사고로 인한 상완골 근위부 골절 수술(OP) 후 영조물 배상책임 공제로 손해사정된 후유장해 케이스.
* [발목 삼복사 골절 케이스 (CASE_004)](ankle-fracture.md) - 좌측 발목 삼복사 골절(S82.830)로 한화손보 상대 배상책임 손해사정(사정금액 약 1.03억)이 이뤄진 케이스.
