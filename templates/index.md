# 템플릿

`wiki/templates/`에서 2026-07-13에 채택(adopt)한 문서 세트. 이 디렉터리가
1차 소스이며, 향후 프로젝트 단의 변경은 여기서 이루어진다 (각 파일의
`provenance` 노트 참고). wiki 쪽 원본은 별도 세션에서 따라잡아 갱신한다.

* [손사서 초안](draft-report.md) - 정답지 실제 양식 기반 손해사정서 초안 템플릿(변형 A 배상책임·후유장해형 I~VII / 변형 B 진단·수술비형 I~VI). 정답지 원문은 초안 작성 input 금지 경고 포함.
* [스크리닝 리포트](screening-report.md) - 케이스 1차 스크리닝 결과 7개 섹션 템플릿.
* [반박 포인트 리포트](rebuttal-points.md) - 감액사유별 반박 논거 출력 형식.
* [금지 표현 가이드](forbidden-expressions.md) - 초안에서 피해야 할 표현과 대체 표현.
* [컴포넌트 표준 출력 계약](component-output.md) - 모든 파이프라인 컴포넌트 출력의 공통 필드 정의.
