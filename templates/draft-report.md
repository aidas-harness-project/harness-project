    - 특별약관상 질병 정의 및 진단확정 방법(병력·신경학적 검진·영상검사 등) 인용
    - **의료자문 결과 인용** — `{자문일} 시행한 {진료과} 의료자문 결과에 따르면 … {소견}` 형태로, 단정 회피 어투 사용
  - `다. 보험금 지급책임` — `[별첨 {번호}] 진단서상 … 최종진단이 확인되어 … 지급책임이 발생함`
- **2. 면책사유 검토** — R04(지급요건), R05(면책), R15(면책·감액기간), R20(계약 전 알릴의무)처럼 약관상 지급책임 또는 면책 여부에 관한 사유를 이 절에 반영
  - `가.` 보통약관 [보험금을 지급하지 않는 사유] 조항 인용
  - `나.` 본 건 면책 해당 여부 판단(`… 해당 사항이 없으므로 지급책임은 존재함` 또는 반대 취지)

## V. 보험금 사정
- **1. 보험금 사정내역** — 담보별 금액. 담보 2개 이상이면 소계 구조(`가./나./다. 합계`)
- **2. 보험금 사정금액** — `一金 {한글금액}整 (₩ {금액})` 강조 표기
- **3. 사정 결과 및 의견** — 줄글(약관·진단서·의무기록·의료자문결과지 근거 적정) + 추가보험금 유보 문단

## VI. 증빙자료
테두리 박스 안 번호 목록(항목명만). 하단 `책임사정사 {사정사명}` + 직인.
항목 골격: `진단서 / 초진기록지 / 경과기록지 / 영상판독지 / 검사결과지 /
의료자문결과지 / 보험증권 사본 / 신분증 및 통장사본 / 위임장 사본`

# 작성 규칙

1. **어투**: 본문 판단은 개조식 종결("~함", "~할 것임", "~하겠음"). 의견·안내부는 존칭("~님", "~바랍니다").
2. **근거 인용 형식**: `[별첨 제{n}호] {자료명}` 또는 `"{의료기관/진단서}상 …"` 패턴으로 문장에 연결한다. 모든 판단 문장은 근거 자료(진단서·의무기록·의료자문·약관·법령)와 연결한다.
3. **단정 회피**: 의료·법률 확정 표현은 완곡한 검토 표현으로 대체한다. 표현 수위는 [금지 표현 가이드](forbidden-expressions.md)를 따른다. 확정판결·재판 시 변동 가능성 단서를 사정 의견부에 명시.
4. **금액 표기**: 아라비아 숫자(천단위 콤마) + 최종 사정금액은 `一金 {한글}整 (₩ {숫자})` 병기. 손해액 합계·사정금액은 백원미만 절사 관례.
5. **계산 서식**: 표 없이 개조식 계산식으로 나열하되 계산 순서(구간 분해 → 소계 → 과실/절사)를 유지. 실제 상수·계수·금액은 파이프라인 초안에서도 입력 산출물에 없는 값은 채우지 않는다(harness-guardrails P3, [draft-report agent](../.claude/agents/draft-report.md)의 작성 원칙).
6. **법령·규정 인용**: 보험업법 제188조, 보험업감독규정 제9-18조·제9-21조, (변형 A) 민법 제750·758조·상법 제724조 등은 서식 고정 보일러플레이트로 유지(케이스와 무관하게 동일 — 정답지 고유값이 아님).
7. 검수는 [critic agent](../.claude/agents/critic.md)가 수행한다.
